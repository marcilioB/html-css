Entendo o uso do target e rel

Os atributos target e rel são usados para controlar o comportamento de um link quando ele é clicado.

    O atributo target especifica o contexto de navegação em que o link deve ser aberto. Os valores possíveis são:
  • _self: abre o link na mesma guia ou janela que o link atual.
  • _blank: abre o link em uma nova guia ou janela.
  • _parent: abre o link no contexto de navegação pai do link atual.
  • _top: abre o link no contexto de navegação superior do link atual.

O atributo rel é usado para especificar a relação entre o link atual e o recurso vinculado. Ele pode ser usado para fornecer informações aos usuários, navegadores e mecanismos de pesquisa.

    Os valores possíveis para o atributo rel são definidos por padrões específicos. Alguns dos valores mais comuns são:

  • alternate:
    •  indica que o link é um link alternativo para o mesmo conteúdo.
  • author:
    •  indica que o link é para o autor do recurso vinculado.
  • canonical:
    •  indica que o link é a versão canônica do recurso vinculado.
  • copyright:
    •  indica que o link é para o proprietário dos direitos autorais do recurso vinculado.
  • help:
    •  indica que o link fornece ajuda para o recurso vinculado.
  • next:
    •  indica que o link é para a próxima página do recurso vinculado.
  • nofollow:
    •  indica que o link não deve ser seguido pelos mecanismos de pesquisa.
  • prev:
    •  indica que o link é para a página anterior do recurso vinculado.
  • search:
    •  indica que o link é para um mecanismo de pesquisa.
